# Tetris
pygame을 이용한 간단한 tetris를 만들었습니다.

## 실행 
tetris_ms.py 파일을 실행시켜주면 게임이 실행됩니다.

## 조작법
→ : 우로 이동  
← : 좌로 이동  
↓ : 아래로 이동  
Space bar : 회전  
Z : 하드드롭
